The purpose of this dataset was to analyse a given dataset in SQL and create some visualisations and a presentation. Due to the size and structure of the dataset, it was impossible to create a funnel with the raw data so I had to aggregate it in SQL and then export the aggregated data into Tableau for the funnel analysis.
